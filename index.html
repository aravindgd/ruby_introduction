Ability to easily build *DSL

Example of DSL: 
						
resources :consulting_location_doctor_schedules
namespace :admin do
	resources :appointments, except: [:new, :create, :edit, :update] do
		get :cancel_appointment, on: :member
		get :users, on: :collection
		get :doctors, on: :collection
		get :confirm_appointment, on: :member
	end
end



*Domain Specific Language(DSL) is “a programming language of limited expressiveness focused on a particular domain”. A way to express a logic specific to a particular domain without needing to write too much extraneous code

class with inheritance

Inheritance is acheived with `child_class < parent_class` syntax 


class Calculator
	def add
		puts @var1+@var2
	end
end
class ComplexCalculator < Calculator
	def initialize(var1, var2)
		@var1 = var1
		@var2 = var2
	end
end
test = ComplexCalculator.new(1,2)
test.add

OUTPUT:
3

Modules are simply a set of code that can be attached to any other file in ruby

They are not initialized, they are just included inside another class or Module

What is a web application framework?

A web application framework (WAF) is a software framework that is designed to support the development of dynamic websites, web applications, web services and web resources. The framework aims to alleviate the overhead associated with common activities performed in web development. For example, many frameworks provide libraries for database access, templating frameworks and session management, and they often promote code reuse.
